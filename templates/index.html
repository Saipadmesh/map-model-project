Welcome

This is a demo application to predict the shortest route between two junctions. This application can also be used to find the nearest landmark like Hospital, Petrol Station and more. This application uses neo4j as a database, and uses flask for connectivity.

This application uses Bokeh to draw a node graph which shows the junctions and the roads connecting them. Click on the "Open Map" button to proceed further. For more options, use the button present on the top right of the navbar.
